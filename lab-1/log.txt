Async / Await explained simply

This part is often confusing — I’ll simplify it.

Problem (why async exists)
Imagine:

Thread.Sleep(3000);

Program FREEZES for 3 seconds

Solution: async/await
Allows:
“Do something in background while program continues”

Example
public static async Task GetDataAsync()
{
    Console.WriteLine("Start fetching data...");
    
    await Task.Delay(2000); // simulate database/API
    
    Console.WriteLine("Data fetched!");
}



Prompt:
objasni async i await detaljno



Response:
Neke operacije traju dulje — čitanje iz baze, API poziv, čitanje filea. Bez asynca, program blokira cijeli thread i ne može raditi ništa drugo dok čeka.
// SINKRONO - thread je blokiran, ništa se ne izvodi dok čeka
var timovi = DohvatiIzBaze(); // čeka 2 sekunde, program "zamrznut"
Console.WriteLine("Ovo čeka 2 sekunde");

// ASINKRONO - thread nije blokiran
var timovi = await DohvatiIzBazeAsync(); // čeka 2 sekunde, ali thread slobodan
Console.WriteLine("Ovo se ispiše tek kad gornji završi, ali thread nije blokiran");

Kako se piše
Tri pravila koja uvijek vrijede:
1. Metoda koja koristi await mora imati async u potpisu
2. Povratni tip mora biti Task ili Task<T>
3. await stavljaš ispred poziva koji traje dulje
// Bez povratne vrijednosti → Task
async Task IspisiBrzuPorukunAsync()
{
    await Task.Delay(1000);
    Console.WriteLine("Gotovo!");
}

// S povratnom vrijednošću → Task<T>
async Task<string> DohvatiImePobjednikaAsync(int raceId)
{
    await Task.Delay(500); // simulacija upita prema bazi
    return "Leclerc";
}

// Pozivanje
await IspisiBrzuPorukunAsync();
string pobjednik = await DohvatiImePobjednikaAsync(1);

Paralelno vs sekvencijalno
Ovo je najvažnija razlika za razumjeti:
// SEKVENCIJALNO - čeka jedan pa drugi, ukupno 3 sekunde
var a = await PrviFunkcijaAsync();  // čeka 1s
var b = await DrugaFunkcijaAsync(); // čeka 2s
// ukupno: 3 sekunde

// PARALELNO - oba kreću istovremeno, ukupno 2 sekunde
var taskA = PrviFunkcijaAsync();  // samo pokrene, ne čeka
var taskB = DrugaFunkcijaAsync(); // samo pokrene, ne čeka
var a = await taskA;              // sad čeka
var b = await taskB;              // već gotovo ili skoro
// ukupno: 2 sekunde (koliko traje duži)
